
Quick Tutorial for experienced functional programmers
=====================================================

Disco is a small yet expressive, pure functional programming language
designed especially to be used in the context of a discrete
mathematics course.  Right now it is in a rough prototype stage; this
tutorial is only useful for developers who already have some knowledge
of functional programming.
