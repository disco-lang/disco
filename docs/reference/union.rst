Set operations
==============

The *union* or *intersection* of two :doc:`finite sets <set>` can be found using
the ``union`` and ``intersect`` operators, or using the Unicode
notation ``∪`` and ``∩``.

::

   Disco> {1,2,3} union {2,3,4}
   {1, 2, 3, 4}
   Disco> {1,2,3} intersect {2,3,4}
   {2, 3}
